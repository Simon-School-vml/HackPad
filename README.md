# BOM

for this very lovely keypad you need a few components.

1. PCB custom made
2. Xiao RP2040
3. 16x MX style switches (any of your flavor)
4. 16x Keycaps
5. 16x diode 1n4148
6. 16x SK6812 MINI-E LEDs
7. 0.91 inch OLED display

M3 mounting hardware
4x M3x22-25 (for case)
3x M3x6     (for PCB)

3D printed parts
the four parts from the step file
